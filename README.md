# Field-Supported BVLOS Compliance Kit

This repository contains reusable templates for SORA-guided SAIL II field-supported BVLOS UAS operations.

The templates are intended to help operators structure evidence for roles, training, competency, maintenance, inspection, emergency response, occurrence reporting, human factors, and OSO traceability. They are provided as a starting point only. They do not demonstrate compliance by themselves and do not replace an operation-specific risk assessment, competent review, or regulatory approval.

## Scope

Current scope:

- SORA-guided SAIL II operations.
- Field-supported BVLOS operations, where the crew is physically present in or near the operating area.
- Template-based evidence management for short-duration field operations.

Out of scope:

- Higher SAIL levels.
- Remote BVLOS operations with geographically remote crew.
- Jurisdiction-specific legal advice.
- A complete SORA application.
- Any claim that the templates automatically satisfy authority requirements.

## How to use the templates

1. Copy the files into a separate working folder for your operation.
2. Replace all bracketed placeholders, such as `[Operator Name]`, `[Operation Name]`, `[Aircraft Type]`, and `[Operating Area]`.
3. Adapt each template to your aircraft, crew structure, operating area, approval conditions, and authority expectations.
4. Complete the SAIL II OSO compliance matrix using your own evidence. Do not copy the example status values as a compliance conclusion for another operation.
5. Build the field data-capture forms from the form specification document and response schema workbook.
6. Review all completed documents before use in the field.
7. After the operation, check the OSO matrix again and record any remaining evidence gaps.

## Recommended workflow

1. Start with `00_Operations_Manual_Template.docx`.
2. Define roles using `Annex_A_Roles_and_Responsibilities_Template.docx`.
3. Set up training and competency tracking using Annex B and the competency matrix.
4. Prepare maintenance and inspection evidence using Annex C and the inspection checklists.
5. Prepare emergency response and occurrence reporting using Annex D and Annex E.
6. Complete the human factors and HMI review using Annex F.
7. Use `00_SAIL_II_OSO_Compliance_Matrix_Template.xlsx` to link each OSO to the evidence generated by the templates.
